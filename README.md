# NMP-Server
Portable Nginx, Mysql, PHP development stack for Windows.

Tested on Windows XP/Vista/7/8.1/10.

Features
--------
- Nginx 1.9.6
- MySQL 5.6.27
- PHP 5.6.15

Usage
-----
Run nmp.exe to start the process and launch NMP-Server.
